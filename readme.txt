Tienda en construcción – Cristian Rodriguez

LOGIN:
•	Objetivo cumplido: Manejo de usuarios registrados y autenticación.
•	Uso de Objetos: Clase Usuarios para representar usuarios registrados.
•	Uso de Variables y Estructuras de Control: Variables para capturar datos de usuario y contraseña, control de flujo con if-else.
•	Almacenamiento Local: LocalStorage para almacenar y recuperar usuarios registrados y la sesión activa.
PANEL ADMINISTRATIVO:
•	Objetivo cumplido: Mostrar usuarios y administradores, permitir cambios de estado.
•	Uso de Objetos: Clase Usuarios para representar usuarios.
•	Manipulación de Arrays: Métodos como forEach para iterar sobre los usuarios y push para agregar nuevos usuarios.
•	Interactividad y Eventos: Agrega eventos para cambiar el estado de administrador y actualiza el almacenamiento local.
REGISTRO DE USUARIOS:
•	Objetivo cumplido: Registro de nuevos usuarios.
•	Uso de Objetos: Clase Usuarios para crear nuevos usuarios.
•	Manipulación de Arrays: Agrega nuevos usuarios al array usuariosRegistrados.
•	Almacenamiento Local: Guarda los usuarios registrados en localStorage.
SESIÓN:
•	Objetivo cumplido: Verificación de sesión activa y redirección según estado.
•	Uso de Variables y Estructuras de Control: Verifica localStorage para la sesión activa y decide la redirección.
•	Manipulación de URLs: Utiliza window.location.href para redirigir según la sesión y la página actual.
TIENDA (STORE):
•	Objetivo cumplido: Interacción con productos y manejo de carrito.
•	Uso de Objetos: Clase Producto para representar productos.
•	Manipulación de Arrays: Crea y filtra productos, actualiza el carrito con botones de agregar y quitar.
•	Interactividad y Eventos: Agrega eventos a botones para modificar cantidades y calcular total.
MAIN (INICIO):
•	Objetivo cumplido: Creación de productos y filtros básicos.
•	Uso de Objetos: Clase Producto para crear productos.
•	Manipulación de Arrays: Crea un array de productos y filtra según términos de búsqueda.
•	Interfaz de Usuario: Renderiza productos dinámicamente y responde a eventos de búsqueda.
